# INSTRUCTIONS TO RUNNING IS424: HDB RESALE PRICE PREDICTION PROJECT
1. Install Anaconda from https://anaconda.org/anaconda/python
2. Open Anaconda Prompt, and install sklearn and statsmodels using the following commands:
	- pip install sklearn
	- pip install xgboost
3a. Download the appropriate XGBoost distribution from https://www.lfd.uci.edu/~gohlke/pythonlibs/
3b. From Anaconda Prompt, navigate to the downloaded distribution
3c. Type the following command:
	- pip install <filename>
4. Launch Anaconda and subsequently Jupyter Notebook
5. Upload the following files:
	- script.ipynb
	- resale-flat-prices.xlsx
	- addresses.xlsx
	- station-mappings.xlsx
	- macro data.xlsx
6. Open script.ipnyb
7. Click on "Cell" -> "Run All" from the toolbar
